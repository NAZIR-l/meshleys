## Description

AWS Cognito integration with Nest.js.
Implements Cognito Sign In, Sign up and Confirmation

# Setup

Make sure you have your AWS Cognito user pool credentials as specified on `.env.example` file

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
