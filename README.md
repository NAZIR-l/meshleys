# AWS Cognito with Nest.js Integration

## Description

This repo show you how you can use AWS Cognito with Nest.js

## Env' Setup

```bash
touch .env
```
After you have created the .env file add the `env` variables from you AWS Cognito settings

```javascript
COGNITO_USER_POOL_ID=
COGNITO_CLIENT_ID=
COGNITO_REGION=
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
